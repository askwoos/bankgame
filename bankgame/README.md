# 이름 - 은행게임

![Alt text](bank.jpg)


## 기능개요 

- 예금을 하면 신용등급이 올라감 
- 초기 등급은 10등급
- 한번 예금할때만다 1등급이 올라감

## 다형성개요 

- MainBank interface를 만들고 KakaoBank,Toss,Kbank가 
- MainBank에서 withDraw() ,deposit(), getBalance()
- KakaoBank,Toss,Kbank 가 @Override한다.
